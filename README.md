# AI-Powered-Smart-Water-Footprint-Optimizer
##  Project Overview

This project is a Machine Learning-based web application that predicts the **water footprint** based on household water usage activities.

Users can input daily water usage values such as bathing, cooking, washing, gardening, and drinking, and the app predicts the total water footprint along with smart recommendations.


## Model Used

* Random Forest Regressor
* Built using Scikit-learn

##  Features Used

* Bathing (L)
* Cooking (L)
* Washing (L)
* Gardening (L)
* Drinking (L)
* Total Usage (calculated feature)

* Reduce shower time
* Use efficient washing methods

##  How to Run Locally

1. Clone the repository
2. Install dependencies:

```
pip install -r requirements.txt
```

3. Run the app:

```
streamlit run app.py
```
